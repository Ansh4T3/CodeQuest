# CodeQuest

**An Online Coding Practice Platform**

---

## 📌 Project Overview

CodeQuest is a LeetCode-like platform that allows:
- **Admins** to add coding problems.
- **Users** to register, view problems, submit solutions, and get pass/fail feedback.

---

## ⚙️ Tech Stack

- **Frontend:** React JS
- **Backend:** Django REST Framework
- **Database:** MongoDB
- **Version Control:** Git & GitHub

---

## 📂 Folder Structure

/CodeQuest/
├── backend/ # Django backend
├── frontend/ # React frontend


---

## ✅ Key Features

- User authentication (register/login)
- Problem list & detail view
- Submission system
- Solution check (basic)
- Admin panel for problem management
- MongoDB for storing problems, users, submissions
- Git & GitHub version control
